# Library Inventory Manager

A Python-based CLI application that uses Object-Oriented Programming (OOP) to manage a library's book collection with persistent storage.

## Features
- **OOP Architecture:** Built using `Book` and `LibraryInventory` classes.
- **Persistent Storage:** Automatically saves and loads data using `library.json`.
- **Circulation Management:** Issue and Return books, tracking their status in real-time.
- **Search:** Filter books by title or look up by ISBN.

## Usage

1. **Run the application:**
   ```bash
   python main.py
2. **Menu Options:**

Add Book: Register a new book.

List All: View inventory and status.

Issue/Return: Manage book availability using ISBN
